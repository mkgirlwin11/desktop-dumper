# Desktop Dumper

**Version:** 1.0  
**Developer:** mkgirl  

A lightweight Windows batch tool that creates timestamped ZIP backups of your Desktop folder with system information.

## Features

- Copies all files and folders from the Desktop (excluding the tool's own directory)
- Generates a `backup-info.txt` with session metadata
- Compresses everything into a versioned ZIP archive
- Uses only built-in Windows components (no third-party tools required)

## Requirements

- Windows 7 or later (WMIC, Robocopy, and PowerShell must be available)
- .NET Framework 4.5+ (for `System.IO.Compression.ZipFile`)

## Usage

Double-click `desktop-dumper.cmd` or run from the command line:

```
desktop-dumper.cmd
```

## Output

Archives are stored in the `backups\` subdirectory with the naming format:

```
desktop-backup_YYYY-MM-DD_HHMMSS.zip
```

### Contents of each archive

| Item | Description |
|------|-------------|
| `backup-info.txt` | Timestamp, desktop size, OS version/build, username |
| `*` | All Desktop files and folders |

### Example `backup-info.txt`

```
========================================
 Desktop Backup Information
 Generated by Desktop Dumper v1.0
 Developer: mkgirl
========================================

Time: 14:30:45
Date: 2026-06-11

Desktop Path: C:\Users\jdoe\Desktop
Total Desktop Size: 1,234,567 bytes

Operating System: Microsoft Windows 10 Pro
OS Version: 10.0.19045
OS Build: 19045

Username: jdoe
Computer Name: DESKTOP-ABC123

Backup Timestamp: 2026-06-11_143045
========================================
```

## Notes

- The tool skips its own folder during copy to prevent recursive nesting
- Hidden and system files are included in the backup
- Temporary backup folder is automatically deleted after ZIP creation
